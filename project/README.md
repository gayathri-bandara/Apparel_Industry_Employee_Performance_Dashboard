Goal:
Show how employees perform in a garment factory based on:
productivity / attendance / quality of work / overtime

# 👕 Apparel Industry Employee Performance Dashboard

## 📊 Project Overview

This project presents a Power BI dashboard that analyzes employee performance in an apparel manufacturing environment.

The dashboard focuses on:

- Productivity (Units Produced)
- Quality (Defects)
- Attendance
- Overtime

---

## 🛠 Tools & Technologies

- Power BI
- Python (Pandas, NumPy, Faker)
- VS Code
